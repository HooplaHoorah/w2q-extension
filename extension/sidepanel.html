How to use

  1. Select text on any page.
  2. Right-click → Send selection → Web-to-Quest (or paste into Input).
  3. Use the buttons below (Summarize/Rewrite/Translate/Extract).
  4. Note: Summarize/Rewrite require on-device AI (Gemini Nano). If unavailable, those buttons stay disabled. Translate/Extract work everywhere.
